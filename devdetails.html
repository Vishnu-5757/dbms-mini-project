About Project

The police data Management system is an application that allows all the paper work that are done to be stored in system digitaly with super ease. If there is a need of some informaton of a particular criminal which is very urgent and necessary then you need to search in huge heap of files and that takes time and also wastes effort. Hence to overcome these problems and for the ease of cops, the police data Management system is useful. Henc cops not only save time and effeciency but also reduce paper work.

Police data management application will surely help in storing the records related to the criminals, crime type, crime status and so on. This can allow a person to enter or delete the recors if necessary. This application not only records the criminals or the offenders but also holds the record of officers and also the department they belong to.

As far the security, only the authorized users or tge officers will have the access to the system. In brief this application can help in getting the information of the criminals of many years back. This can also help in minimizing almost all of the work of the police.
